# Quick Start Guide - Contract Law Academy PWA

Get your learning app up and running in minutes!

## ⚡ 5-Minute Quick Start

### Step 1: Choose Your Hosting (Pick ONE)

**EASIEST - Netlify (Recommended)**
1. Go to https://app.netlify.com
2. Click "Add new site" → "Import an existing project"
3. Connect GitHub and select your repository
4. Click Deploy
5. Done! ✅

**OR - Vercel**
1. Go to https://vercel.com
2. Click "New Project"
3. Import your GitHub repository
4. Click Deploy
5. Done! ✅

**OR - GitHub Pages**
1. Create GitHub repository
2. Enable "Pages" in settings
3. Select main branch as source
4. Done! ✅

### Step 2: Access Your App

After deployment, you'll get a URL like:
- Netlify: `https://your-app-name.netlify.app`
- Vercel: `https://your-app-name.vercel.app`
- GitHub: `https://username.github.io/repo-name`

### Step 3: Install on Your Phone

**iPhone/iPad:**
1. Open link in Safari
2. Tap Share button
3. Select "Add to Home Screen"
4. Tap "Add"

**Android:**
1. Open link in Chrome
2. Tap menu (⋮)
3. Tap "Install app"
4. Tap "Install"

**Done! 🎉 Your app is ready to use!**

## 🔧 Common Issues & Fixes

### "Can't install the app"
- Make sure you're using Chrome/Safari
- Check that HTTPS is enabled on hosting
- Reload the page
- Clear browser cache

### "Audio not playing"
- Check browser allows audio (not muted)
- Verify audio files are uploaded
- Try different browser
- Check file names match in app.js

### "Service worker won't register"
- Ensure HTTPS is enabled
- Close all browser tabs and reopen
- Clear cache and reload
- Check browser console for errors

### "Progress not saving"
- Disable private/incognito mode
- Check available storage space
- Try different browser
- Clear cache and reload

**Still stuck?** See DEPLOYMENT.md for detailed troubleshooting.

## ❓ FAQ

**Q: Is this app free?**
A: Yes! The app is completely free to use and deploy.

**Q: Can I modify the content?**
A: Yes! Edit the HTML, CSS, and JavaScript as needed.

**Q: Does it work offline?**
A: Yes! Once installed, the app works without internet.

**Q: Can I add more lessons?**
A: Yes! Edit the `lessonsData` array in app.js.

**Q: Is my data secure?**
A: Yes! All data stays on your device. Nothing is sent to servers.

**Q: Which phones are supported?**
A: iPhone (Safari) and Android (Chrome) with iOS 11.3+ or Android 5+.

**Q: Do I need a backend server?**
A: No! This is a fully client-side app. Any web hosting works.
