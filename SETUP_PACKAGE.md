## ⚙️ Customization Guide

### Change Lesson Content
Edit `app.js` in the `lessonsData` array:
```javascript
{
  id: 1,
  title: "Your Title",
  icon: "📚",
  duration: "10-12 min",
  description: "Your description",
  file: "audio_filename.wav",
  captions: [
    { time: "0:00", text: "Caption text" }
  ]
}
```

### Change Colors
Edit CSS variables in `index.html`:
```css
--primary: #2563eb;      /* Main blue */
--secondary: #7c3aed;    /* Purple */
--success: #10b981;      /* Green */
--warning: #f59e0b;      /* Orange */
```

### Change Guidelines
Edit HTML in `index.html` in the "Guidelines Tab" section.

### Add New Features
Edit `app.js` and `index.html` to add functionality.

## 🆘 Troubleshooting

### App Won't Load
- ✅ Check internet connection
- ✅ Verify all files uploaded
- ✅ Clear browser cache
- ✅ Try different browser
- ✅ Check HTTPS is enabled

### Audio Won't Play
- ✅ Ensure audio files are uploaded
- ✅ Check file names match app.js
- ✅ Verify MIME type is audio/wav
- ✅ Use relative paths like "./filename.wav"
- ✅ Try different browser

### Service Worker Won't Register
- ✅ Ensure HTTPS is enabled
- ✅ Check sw.js file exists
- ✅ Clear cache and reload
- ✅ Check browser console for errors
- ✅ Use Chrome for best results

### Progress Not Saving
- ✅ Disable private/incognito mode
- ✅ Enable LocalStorage in browser
- ✅ Check available storage space
- ✅ Clear cache
- ✅ Try different browser

### Can't Install on Phone
- ✅ Use Chrome (Android) or Safari (iOS)
- ✅ Ensure HTTPS is enabled
- ✅ Reload page
- ✅ Wait 30 seconds for PWA detection
- ✅ Check manifest loads (browser console)
